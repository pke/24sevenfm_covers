24seven.fm Covers - desktop viewer
==================================

A standalone desktop app that shows the current cover art for the 24seven.fm
radio stations (StreamingSoundtracks, 1980s.FM, Adagio.FM, Death.FM,
Entranced.FM), GPU-rendered with Direct2D - with an optional remaining-time
countdown and crossfade / flip transitions. No music player required.

RUN
---
Run  24sevenfm_covers.exe  - it is self-contained (no install, no runtime).
On first launch it asks which station to show; pick one and the cover appears.

OPTIONS
-------
Right-click the window (or its system menu / Alt+Space) > Options...
  - Station : which 24seven.fm station to display
  - Overlay : remaining-time countdown + size
  - Transition + duration, rolling digits
Double-click the window (or Esc) toggles fullscreen.

UNINSTALL
---------
Delete 24sevenfm_covers.exe (and the 24seven.fm-covers.ini next to it).

Requires Windows 7 or later (Direct2D). 64-bit.
